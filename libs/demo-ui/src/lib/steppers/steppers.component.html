Steppers

How to use

Stepper can be implemented with Angular binding framework or by itself without binding.

Component Directive for Stepper

rex-stepper

Available parameters

Angular binding parameters

type (Optional): Type of the stepper. Possible values: StepperType.Default | StepperType.Horizontal

steps (Optional): Steps in the stepper as Array of StepperStep class. When not defined, nothing will be displayed.

Static value parameters

data-type (Optional): Type of the stepper. Possible values: "" | "horizontal"

data-steps (Optional): Steps in the steps as string format of Array of StepperStep class. When not defined, nothing will be displayed.

Definitions

Enum: StepperType

Package: @rex-lib/one-ui

Properties

Default [Static value: '']: Vertical stepper style

Horizontal [Static value: 'horizontal']: Horizontal stepper style

Enum: StepperStepState

Package: @rex-lib/one-ui

Properties

Default [Static value: '']: Default state where step is not clickable. Indicator is a bordered circle. Line connecting succeeding step will be gray colour.

NextStep [Static value: 'next-step']: Next step state where step is clickable. Indicator is a filled circle with primary colour. Line connecting succeeding step will be primary colour.

Complete [Static value: 'complete']: Complete state where step is clickable. Indicator is a filled circle with primary colour. Line connecting succeeding step will be primary colour.

Class: StepperStep

Package: @rex-lib/one-ui

Attributes

id [Required, string]: Id for the step

pill [Optional, number or string]: Letter or a number to indicate the step. Will display blank if not defined.

icon [Optional, string]: Class(es) for the icon. Will display blank if not defined. If pill is defined, it will take precedence over icon.

title [Required, string]: Title for the step

desc [Optional, string]: Description of the step. It will appear below the title.

state [Optional, StepperStepState]: State of the step. Default value: Default

valid [Optional, boolean]: Indicates if the step is valid. Default value: false

active [Optional, boolean]: Indicates if the step is active. Default value: false
